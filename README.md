# Expense Tracker Application (Python CLI)

A simple command-line Expense Tracker built using Python.  
It helps users track daily expenses and calculate total spending with persistent JSON storage.

---

## Features

- Add expenses with title and amount
- View all expenses
- Delete expenses
- Search expenses by keyword
- View total spending
- Persistent storage using JSON file
- Automatic date tracking

---

## Technologies Used

- Python 3
- JSON (data storage)
- File Handling
- datetime module
